# whatsapp-status-video-optimize
untuk upload video ke status whatsapp agar tidak blur/pecah

## installasi ffmpeg (ubuntu)

1. install ffmpeg
```
sudo apt install ffmpeg -y
```
3. lalu tambahkan izin eksekusi
```
chmod +x alat.sh
```
4. eksekusi file alat.sh
```
./alat.sh
```
